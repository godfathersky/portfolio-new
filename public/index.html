Umiejętności i kompetencje
Umiejętności techniczne Kompetencje miękkie
HTML5 HTML5 to język znaczników używany do strukturyzacji i prezentacji dokumentów hipertekstowych w Internecie. Jest piątą i ostatnią ważną wersją HTML, obecnie emerytowaną rekomendacją W3C. HTML5 ma na celu poprawę języka, zachowanie czytelności dla ludzi i spójności dla komputerów.
CSS3 CSS3 to język opisujący sposób prezentacji graficznej elementów na stronie internetowej. W połączeniu z językiem HTML5 tworzy środowisko umożliwiające stworzenie rozbudowanych i atrakcyjnych stron internetowych.
RWD Responsive Web Design (RWD) to podejście projektowe, które umożliwia tworzenie stron internetowych, które wyglądają dobrze na różnych urządzeniach. Dzięki HTML i CSS strony automatycznie dostosowują się do różnych rozmiarów ekranów i widoków.
Tailwind CSS Tailwind CSS to framework CSS, który pozwala na szybkie tworzenie nowoczesnych stron internetowych, bez konieczności opuszczania kodu HTML. Zamiast tworzenia oddzielnych plików CSS, w Tailwindzie korzysta się z klas bezpośrednio w kodzie HTML.
Bootstrap Bootstrap to najpopularniejszy framework HTML, CSS i JavaScript do tworzenia responsywnych i przyjaznych dla urządzeń mobilnych stron internetowych. Jest darmowy do pobrania i użytkowania.
JavaScript JavaScript to najpopularniejszy język programowania na świecie. Jest używany głównie do tworzenia interaktywnych stron internetowych.
C# C# to nowoczesny, innowacyjny i wieloplatformowy język programowania. Jest używany do tworzenia aplikacji na platformie .NET.
RestAPI REST API (Representational State Transfer) to styl architektoniczny wykorzystywany do projektowania interfejsów API za pomocą protokołu HTTP.
MySQL MySQL to popularny system zarządzania bazą danych (DBMS), który znajduje szerokie zastosowanie w tworzeniu aplikacji internetowych.
SQL SQL (Structured Query Language) to strukturalny oraz deklaratywny język zapytań. Jest używany do tworzenia, modyfikowania relacyjnych baz danych oraz do umieszczania i pobierania danych z tych baz. Decyzję o sposobie przechowywania i pobrania danych pozostawia się systemowi zarządzania bazą danych (DBMS).
Git Git to wolny i otwarty system kontroli wersji, który umożliwia skuteczne zarządzanie projektami o różnej skali.
GitHub GitHub to najważniejsza platforma dla programistów, umożliwiająca rozwijanie oprogramowania, współpracę i zabezpieczanie projektów.
Figma Figma to platforma, która umożliwia zespołom projektowanie, prototypowanie, rozwijanie i zbieranie opinii na temat produktów w jednym miejscu. Oferuje narzędzia do współtworzenia, wyrównywania i standaryzacji systemów projektowych, a także nową przestrzeń dla programistów do tłumaczenia projektów na kod. Figma działa w chmurze, co ułatwia dostęp do projektów z dowolnego miejsca i urządzenia.
Postman Postman to platforma dla programistów, która ułatwia testowanie, projektowanie, dokumentowanie i zarządzanie interfejsami API.
